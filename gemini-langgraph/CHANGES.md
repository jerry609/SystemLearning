# 课程调整说明

## 主要变化

### ✅ 保留的内容
- LangGraph 核心概念和基础
- Gemini API 集成
- 智能体状态管理
- 工具调用和函数
- Web 研究智能体
- FastAPI 后端开发
- 反思与规划模式
- 多智能体系统
- 持久化与内存管理

### ❌ 移除的内容
- React 前端开发（Lab 06）
- 前端流式传输和实时更新（Lab 08 前端部分）
- Node.js 和 npm 依赖
- 所有前端相关的示例和练习

### 🔄 调整的内容

**从 15 个模块减少到 10 个模块**

原课程结构：
- Labs 1-5: 基础
- Labs 6-11: 全栈（包含前端）
- Labs 12-14: 生产
- Lab 15: 综合项目

新课程结构：
- Labs 1-3: 基础
- Labs 4-5: 工具与搜索
- Labs 6-8: 后端与高级模式
- Labs 9-10: 生产与综合

**Lab 重新编号**：
- 原 Lab 07 (FastAPI) → 新 Lab 06
- 原 Lab 09 (反思与规划) → 新 Lab 07
- 原 Lab 10 (多智能体) → 新 Lab 08
- 原 Lab 11 (持久化) → 新 Lab 09
- 原 Lab 15 (综合项目) → 新 Lab 10

### 🎯 新的课程重点

**聚焦：Deep Search Agent（深度搜索智能体）**

综合项目（Lab 10）将构建一个完整的深度搜索智能体系统，包括：
- 多智能体协作（研究员、分析师、综合者、监督者）
- 深度搜索引擎
- 反思与规划能力
- 持久化与内存
- FastAPI 后端 API
- 生产级部署（Docker）
- 完整的测试套件

### 📊 时间估计变化

- 原课程：76-96 小时（15 个模块）
- 新课程：54-68 小时（10 个模块）
- 减少：约 25-30% 的时间

### 🛠️ 技术栈变化

**移除**：
- React
- Node.js
- Vite
- Tailwind CSS
- 所有前端相关库

**保留/增强**：
- Python + LangGraph
- Google Gemini
- FastAPI
- PostgreSQL + Redis
- Docker
- pytest

## 课程优势

1. **更专注**：纯后端开发，深入 AI 智能体系统
2. **更高效**：减少 25-30% 的学习时间
3. **更实用**：聚焦生产级深度搜索智能体
4. **更完整**：综合项目整合所有核心概念

## 适合人群

- 后端开发者
- AI/ML 工程师
- 想要构建智能体系统的开发者
- 对深度搜索和信息检索感兴趣的开发者
