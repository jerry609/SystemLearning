## 🎯 核心技巧

### 枚举右，维护左
将双变量问题转换为单变量问题：
- 枚举右边的元素
- 在左侧查找满足条件的元素
- 用哈希表维护左侧信息

### 枚举中间
对于多变量问题，枚举中间的变量：
- 自动将其他变量隔离开
- 简化位置关系的判断
- 降低问题复杂度

### 遍历对角线
矩阵对角线的遍历技巧：
- 按对角线分组处理
- 利用坐标关系
- 优化空间和时间复杂度
