# 四、其他

包含各种特殊的二分查找应用和技巧。

## 分类

### 1. 基础变体
- 平方根
- 简单搜索

### 2. 峰值查找
- 一维峰值
- 二维峰值
- 山脉数组

### 3. 旋转数组
- 查找最小值
- 搜索元素
- 有重复元素的情况

### 4. 特殊结构
- 完全二叉树
- 缺失元素
- 有序数组中的单一元素

### 5. 高级应用
- 中位数问题
- 不动点
- 矩阵搜索

## 学习建议

1. 这些题目各有特点，没有统一模板
2. 重点理解每道题的特殊性质
3. 学会将问题转化为二分查找
4. 注意边界条件的处理
