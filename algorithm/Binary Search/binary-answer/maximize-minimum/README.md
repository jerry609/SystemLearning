# §2.5 最大化最小值

本质是二分答案求最大，提升下界，看看能否满足条件。

## 核心思想

> **提升下界，看看能否满足条件。**

## 问题特征

- 题目要求**最大化**某个**最小值**
- 例如：最大化最小距离、最大化最小得分等
- 单调性：下界越低越容易满足条件

## 理解方式

想象你要设定一个标准线（下界）：
- 标准线越高，越难达到（越不满足）
- 标准线越低，越容易达到（越满足）
- 找最高的仍能满足条件的标准线

## 与最小化最大值的对比

| 特征 | 最小化最大值 | 最大化最小值 |
|------|-------------|-------------|
| 目标 | 压低上界 | 提升下界 |
| 形象比喻 | 用盖子压住柱子 | 抬高地板 |
| 单调性 | 上界越高越容易 | 下界越低越容易 |
| 二分类型 | 求最小 | 求最大 ⚠️ |

## 解题步骤

1. **识别要最大化的最小值**
2. **二分这个最小值**（作为下界）
3. **check(下界)**：判断能否让最小值 ≥ 下界
4. **返回最大的满足条件的下界**

## 模板

```python
def maximizeMinimum(...):
    def check(lower_bound):
        # 判断能否让最小值 >= lower_bound
        # 通常需要贪心等技巧
        return can_achieve(lower_bound)
    
    # 二分下界（注意是求最大！）
    left = 下界  # 最小值的最小可能
    right = 上界 # 最小值的最大可能
    
    while left + 1 < right:
        mid = (left + right) // 2
        if check(mid):  # 可以达到
            left = mid  # 尝试更高的标准 ⚠️ 注意这里是更新 left
        else:
            right = mid
    
    return left  # ⚠️ 注意返回 left
```

⚠️ **重要**: 这是求最大的二分写法！

## 典型场景

1. **间隔问题**：最大化最小间隔
2. **分配问题**：最大化最小分配
3. **路径问题**：最大化路径最小边权
