# §2.4 最小化最大值

本质是二分答案求最小，用一个"盖子"（上界）去压住最大值。

## 核心思想

> **用一个盖子（上界）去压住最大值，看看能否压住（check 函数）。**

## 问题特征

- 题目要求**最小化**某个**最大值**
- 例如：最小化路径中的最大边、最小化分割后的最大和等
- 单调性：盖子越高越容易满足条件

## 理解方式

想象有很多柱子，你要用一个水平的盖子压住所有柱子：
- 盖子越低，越难压住（越不满足）
- 盖子越高，越容易压住（越满足）
- 找最低的能压住所有柱子的盖子高度

## 解题步骤

1. **识别要最小化的最大值**
2. **二分这个最大值**（作为上界）
3. **check(上界)**：判断是否能让最大值 ≤ 上界
4. **返回最小的满足条件的上界**

## 模板

```python
def minimizeMaximum(...):
    def check(upper_bound):
        # 判断能否让最大值 <= upper_bound
        # 通常需要贪心等技巧
        return can_achieve(upper_bound)
    
    # 二分上界
    left = 下界  # 最大值的最小可能
    right = 上界 # 最大值的最大可能
    
    while left + 1 < right:
        mid = (left + right) // 2
        if check(mid):  # 可以压住
            right = mid  # 尝试更小的盖子
        else:
            left = mid
    
    return right
```

## 典型场景

1. **分割问题**：最小化分割后的最大和
2. **路径问题**：最小化路径中的最大边权
3. **分配问题**：最小化最大负载
