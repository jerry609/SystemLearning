# §2.3 二分间接值

二分的不是答案本身，而是一个和答案有关的值（间接值）。

## 核心特点

- 不直接二分答案
- 二分一个中间变量
- 通过中间变量推导出答案

## 理解方式

有些题目直接二分答案不好做，但可以二分一个相关的值，然后通过这个值计算出答案。

**例子**: 
- 不直接二分最终答案
- 而是二分一个辅助变量 x
- 然后用 x 来计算或验证答案

## 与常规二分答案的区别

| 特征 | 常规二分答案 | 二分间接值 |
|------|-------------|-----------|
| 二分对象 | 直接二分答案 | 二分中间变量 |
| check 函数 | 验证答案可行性 | 通过间接值推导 |
| 最终结果 | 二分的值就是答案 | 需要通过二分值计算答案 |

## 适用场景

- 答案和某个值有明确的数学关系
- 直接二分答案难以 check
- 存在更容易验证的中间量

## 学习建议

1. 理解题目中答案和其他量的关系
2. 找到合适的间接值
3. 建立间接值和答案的联系
4. 写出基于间接值的 check 函数
