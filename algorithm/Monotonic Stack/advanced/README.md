# §1.2 单调栈进阶

## 📖 学习目标

掌握单调栈的进阶应用，包括：
- 链表中的单调栈应用
- 结合排序和分块的问题
- 树的构建
- 复杂的模式匹配
- 优化到 O(n) 的算法

## 💡 学习建议

这部分题目为**选做**，难度较大，建议：
1. 先完成基础部分的所有题目
2. 对单调栈有深入理解后再挑战
3. 每道题都标注了难度分（LeetCode 周赛难度）
4. 可以按难度分从低到高逐步练习

## 📊 难度分布

- **1500-1800**: 入门进阶（1019, 1673, 768）
- **1800-2100**: 中等进阶（654, 456, 3113, 2866, 1944）
- **2100-2500**: 高级进阶（2454, 1130, 2289, 1776）
- **2500+**: 专家级（2736, 3420）

## 🎯 题目分类

### 数据结构变体
- 链表中的单调栈
- 树的构建

### 算法优化
- O(n) 优化
- 分块处理

### 复杂模式
- 132 模式
- 边界元素问题
- K次操作问题

详见 `problems.md`

## 💭 思维扩展

除了标准的单调栈问题，还包含：
- **962. 最大宽度坡** (1608)
- **3542. 将所有元素变为 0 的最少操作次数** (1890)
- **1124. 表现良好的最长时间段** (1908)

这些题目需要结合单调栈和其他技巧（前缀和、哈希表等）。
