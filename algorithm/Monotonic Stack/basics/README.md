# §1.1 单调栈基础

## 📖 原理讲解

单调栈【基础算法精讲 26】

## 💡 核心思想

单调栈用于解决"寻找下一个更大/更小元素"这类问题。栈中元素保持单调性（递增或递减）。

### 应用场景
- 找到每个元素左侧/右侧第一个更大/更小的元素
- 计算以每个元素为最值的子数组范围
- 矩形面积计算
- 字典序优化

## 🎯 算法模板

本目录提供四种语言的模板实现：
- `template.go` - Go语言实现
- `template.py` - Python实现
- `template.java` - Java实现
- `template.cpp` - C++实现

### 模板说明

`nearestGreater` 函数返回两个数组：
- `left[i]`: nums[i] 左侧最近的严格大于 nums[i] 的数的下标，不存在则为 -1
- `right[i]`: nums[i] 右侧最近的严格大于 nums[i] 的数的下标，不存在则为 n

**变体**：
- 如果求严格小于，将条件 `<= x` 改成 `>= x`
- 如果求大于等于/小于等于，调整比较符号即可

## 📝 复杂度分析

- **时间复杂度**: O(n) - 每个元素最多入栈出栈各一次
- **空间复杂度**: O(n) - 栈的空间
