## 📖 学习路径

### 第一阶段：基础入门
1. 理解单调栈的原理和应用场景
2. 掌握基本模板代码
3. 完成 `basics/problems.md` 中的基础题目

### 第二阶段：进阶提升
1. 完成 `advanced/problems.md` 中的进阶题目
2. 学习思维扩展题目

### 第三阶段：专题突破
1. **矩形专题**：掌握利用单调栈解决矩形面积问题
2. **贡献法专题**：理解如何用贡献法优化计算
3. **最小字典序专题**：掌握单调栈在字典序问题中的应用
