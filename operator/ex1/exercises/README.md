## 使用技巧

### 1. 先理解再实现

不要急于写代码，先理解：
- 为什么需要这个功能？
- 它如何工作？
- 如何与其他部分交互？

### 2. 增量开发

不要一次实现所有功能：
- 先实现最简单的版本
- 运行测试验证
- 逐步添加功能
- 持续测试和重构

### 3. 善用日志

添加详细的日志输出：
- 记录关键步骤
- 输出重要变量
- 帮助调试问题

### 4. 参考但不抄袭

查看参考答案时：
- 理解实现思路
- 学习代码风格
- 但要用自己的方式实现

## Go 语言特性

在练习中你将学习：

### 1. 函数作为一等公民

```go
type Step func() error

func createStep() Step {
    return func() error {
        // ...
    }
}
```

### 2. 闭包

```go
func counter() Step {
    count := 0
    return func() error {
        count++
        return nil
    }
}
```

### 3. 高阶函数

```go
func When(condition func() bool, step Step) Step {
    return func() error {
        if condition() {
            return step()
        }
        return nil
    }
}
```

### 4. 装饰器模式

```go
func Trace(step Step, name string) Step {
    return func() error {
        fmt.Printf("[START] %s\n", name)
        err := step()
        fmt.Printf("[END] %s\n", name)
        return err
    }
}
```
