## 🚀 使用指南

### 对于学习者

**推荐学习路径**:

```
1. 阅读 INDEX.md 或 GETTING_STARTED.md
   ↓
2. 按顺序学习练习 1-5
   ↓
3. 每个练习：
   - 阅读文档（15-20 分钟）
   - 动手实现（60-90 分钟）
   - 对比参考答案（15-20 分钟）
   ↓
4. 完成所有练习
```

**每个练习的学习步骤**:

```bash
# 1. 阅读练习文档

# 3. 查看参考答案
cd ../solutions/ex1/

go run .
```
